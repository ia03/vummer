# Vummer

Vummer is a Discord bot that can run and judge Python scripts using [Judge0.](https://github.com/judge0/api) Check out the Discord server
[here.](https://discord.gg/2NGbScj)

## Install

Dependencies are located in `requirements.txt`

```
pip install -r requirements.txt
```

[Judge0](https://github.com/judge0/api) is also needed.

## Configuration

Create a new file `config.py` with the token variable set to that of your bot.
```
token='TOKEN HERE'
```

## Start

Run `main.py`
```
python3 main.py
```
